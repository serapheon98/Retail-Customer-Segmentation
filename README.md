# Retail Customer Segmentation & Sales Performance

## Overview
Analyzed retail transactions to identify high-value customers, revenue drivers, and purchasing behavior patterns using RFM segmentation and KPI analysis.

## Dataset
UCI Online Retail Dataset  
https://archive.ics.uci.edu/ml/datasets/online+retail

## Tech Stack
Python, Pandas, SQL, Power BI

## Structure
- data: dataset
- notebooks: analysis
- sql: queries
- dashboard: Power BI file
- images: visuals
